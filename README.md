# 项目 README（中文）

本项目已安装以下插件，为开发提供 3D 渲染、图标、样式与动效支持：

- **Three.js**：跨平台 WebGL 3D 库  
- **Lucide**：轻量 SVG 图标库  
- **UnoCSS**：即时按需原子化 CSS 引擎  
- **Vue Bits**：Vue 动效与创意 UI 组件库  
- **GSAP**：高性能 JavaScript 动画库

## Three.js 简介

Three.js 是一个跨平台的 JavaScript 3D 库，用于在浏览器中通过 WebGL 创建并展示交互式三维图形与动画，提供了场景、摄像机、灯光、材质、几何体、加载器等完整生态，便于快速构建 3D 应用与可视化。

官网：<https://threejs.org/>

## Lucide 图标库简介

Lucide 是一个社区驱动的开源图标库，提供干净统一的 SVG 图标集合。图标轻量、可定制、支持按需引入（Tree Shaking），可在多种框架与包管理器中使用，适合现代 Web 应用界面设计。

官网：<https://lucide.dev/>

## UnoCSS 简介

UnoCSS 是即时按需的原子化 CSS 引擎，核心不带偏见，所有工具类通过预设提供，并可自由扩展与自定义。它具备高性能、零依赖、按需生成样式的特性，能与多种构建工具与框架集成。

官网：<https://unocss.dev/>

## Vue Bits 组件库简介

Vue Bits 是面向 Vue 的动效与创意 UI 组件库，提供丰富的动画组件、背景效果与交互元素，轻量且支持按需引入，适合为页面增添动态与个性化体验。

官网：<https://vue-bits.dev/>

## GSAP 动画库简介

GSAP（GreenSock Animation Platform）是高性能的 JavaScript 动画库，提供细粒度的时间线控制、强大的缓动函数以及丰富的插件生态（如 ScrollTrigger、MotionPath 等），适用于 DOM、SVG、Canvas、WebGL/Three.js 等多种场景的动效开发。

官网：<https://greensock.com/gsap/>
